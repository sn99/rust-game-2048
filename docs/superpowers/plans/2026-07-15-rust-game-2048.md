# rust-game-2048 Implementation Plan

> **For agentic workers:** Execute task-by-task. Steps use checkbox syntax.

**Goal:** Faithful classic 2048 in the browser (Leptos CSR) with unit-tested pure logic and GitHub Pages deploy.

**Architecture:** Single crate — pure `game` module (no DOM) + Leptos UI + CSS + Trunk + GH Actions.

**Tech Stack:** Rust 2021, Leptos 0.8 CSR, Trunk, fastrand, web-sys (localStorage/touch), GitHub Actions Pages.
